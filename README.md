# SymbolAlgebra

A **DragonBox-style algebra puzzle game** in Unity — drag cards on a two-sided board to learn algebra through play.

## Quick start

1. Open this folder in **Unity 2022.3+** or **Unity 6**
2. Open `Assets/DragonBoxAlgebra/Scenes/DragonBox.unity`
3. Press **Play**

## Controls

- **Drag cards** together on the same side to combine
- **Drag from hand** onto the board (adds to both sides)
- **↩ Undo** / **⏪ Rewind** (top right)
