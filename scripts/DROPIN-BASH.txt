# Paste into Git Bash (Windows) — SymbolAlgebra drop-ins

## Your Unity folder: C:\Users\rober\SymbolAlgebra

### One command — always use this (prevents duplicate-script errors)

**Easiest — run every time you want latest code in Unity:**

```bash
cd /c/path/to/MarbleMazePan3D
SYMBOL_ALGEBRA_DIR="/c/Users/rober/SymbolAlgebra" bash scripts/update-unity.sh
```

This pulls latest, deletes bad UI duplicates, and imports to the right folders.

**Do not:**
- Manually copy `.cs` files into `Scripts/UI/`
- Commit Unity `.meta` files in GitHub Desktop unless you meant to change the repo
- Run an old `sync-dropins.sh` from before the routing fix

**Gameplay/** = HandVisualRules, ThemeAssignment, AlgebraGameController, etc.  
**UI/** = BoardView, CardWidget, HandView, etc.

---

### Fix compile errors (duplicate HandVisualRules / ThemeAssignment)

If Unity shows **CS0101 duplicate definition** for `HandVisualRules` or `ThemeAssignment`:

```bash
cd /c/Users/rober/SymbolAlgebra
rm -f Assets/DragonBoxAlgebra/Scripts/UI/HandVisualRules.cs
rm -f Assets/DragonBoxAlgebra/Scripts/UI/HandVisualRules.cs.meta
rm -f Assets/DragonBoxAlgebra/Scripts/UI/ThemeAssignment.cs
rm -f Assets/DragonBoxAlgebra/Scripts/UI/ThemeAssignment.cs.meta
rm -f Assets/DragonBoxAlgebra/Scripts/UI/BoardSideLayout.cs.meta
git pull origin cursor/ch1-merge-intro-1c5d
bash scripts/sync-dropins.sh import
```

The import script now routes those files to **Gameplay/** and removes stale UI copies.

