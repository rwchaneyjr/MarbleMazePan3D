### Fix compile errors (duplicate HandVisualRules / ThemeAssignment)

If Unity shows **CS0101 duplicate definition** for `HandVisualRules` or `ThemeAssignment`:

```bash
cd /c/Users/rober/SymbolAlgebra
rm -f Assets/DragonBoxAlgebra/Scripts/UI/HandVisualRules.cs
rm -f Assets/DragonBoxAlgebra/Scripts/UI/HandVisualRules.cs.meta
rm -f Assets/DragonBoxAlgebra/Scripts/UI/ThemeAssignment.cs
rm -f Assets/DragonBoxAlgebra/Scripts/UI/ThemeAssignment.cs.meta
rm -f Assets/DragonBoxAlgebra/Scripts/UI/BoardSideLayout.cs.meta
git pull origin cursor/ch1-merge-intro-1c5d
bash scripts/sync-dropins.sh import
```

The import script now routes those files to **Gameplay/** and removes stale UI copies.

### Update dropins/ folder after you change scripts in Assets

```bash
bash scripts/sync-dropins.sh export
```
