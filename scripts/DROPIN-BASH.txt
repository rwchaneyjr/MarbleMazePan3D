# Paste into Git Bash (Windows) — SymbolAlgebra drop-ins

## Your Unity folder: C:\Users\rober\SymbolAlgebra

### One command — copy latest drop-ins into your Unity project

From a git clone of MarbleMazePan3D (this repo):

```bash
cd /c/Users/rober/SymbolAlgebra
bash scripts/sync-dropins.sh import
```

Or if the repo lives elsewhere:

```bash
SYMBOL_ALGEBRA_DIR="/c/Users/rober/SymbolAlgebra" bash /c/path/to/MarbleMazePan3D/scripts/sync-dropins.sh import
```

---

### Update dropins/ folder after you change scripts in Assets

```bash
cd /c/Users/rober/SymbolAlgebra
git pull origin main
bash scripts/sync-dropins.sh export
```

### Import into this repo's Assets (cloud agent / local dev)
