# Paste into Git Bash (Windows) — SymbolAlgebra drop-ins

## Your Unity folder: C:\Users\rober\SymbolAlgebra

### If scripts/sync-dropins.sh is missing — paste this whole block

Works without any script file (copies dropins/ into Assets):

```bash
cd /c/Users/rober/SymbolAlgebra
S=Assets/DragonBoxAlgebra/Scripts
mkdir -p "$S/Core" "$S/Gameplay" "$S/UI" "$S/Audio"
cp dropins/AlgebraBootstrap.cs "$S/"
cp dropins/AlgebraBoard.cs dropins/BoardCard.cs dropins/BoardSide.cs dropins/CardKind.cs dropins/CombineRules.cs dropins/WinChecker.cs "$S/Core/"
cp dropins/AlgebraGameController.cs dropins/BalancePending.cs dropins/CardFlipRules.cs dropins/GameSnapshot.cs dropins/HandRules.cs dropins/LevelDefinition.cs dropins/LevelGenerator.cs dropins/LevelLibrary.cs dropins/MoveTracker.cs dropins/PendingCancelMarker.cs "$S/Gameplay/"
cp dropins/AudioManager.cs "$S/Audio/"
cp dropins/*.cs "$S/UI/" 2>/dev/null || true
for f in AlgebraBootstrap AlgebraBoard BoardCard BoardSide CardKind CombineRules WinChecker AlgebraGameController BalancePending CardFlipRules GameSnapshot HandRules LevelDefinition LevelGenerator LevelLibrary MoveTracker PendingCancelMarker AudioManager; do
  rm -f "$S/UI/${f}.cs" 2>/dev/null || true
done
echo "Done — open Unity and press Play"
```
